# Darsah Apps - Islamic Super App

Darsah Apps adalah *Islamic Super App* modern berbasis web (PWA-ready) yang dirancang untuk memenuhi kebutuhan ibadah harian umat Islam. Aplikasi ini menggabungkan desain estetika premium yang terinspirasi oleh NU Online dengan fungsionalitas teknologi terbaru.

## ✨ Fitur Utama

### 📖 Al-Qur'an & Ibadah
- **Al-Qur'an Digital**: 114 Surah lengkap dengan teks Arab, Latin, dan Terjemahan Indonesia (API-driven).
- **Jadwal Shalat**: Waktu shalat akurat berdasarkan lokasi pengguna (real-time API).
- **Kompas Kiblat**: Penunjuk arah Ka'bah interaktif menggunakan sensor perangkat.
- **Tasbih Digital**: Penghitung dzikir dengan animasi progress ring dan berbagai pilihan wirid.
- **Doa Harian**: Kumpulan doa-doa harian lengkap dengan kategori.
- **Kalkulator Zakat**: Penghitung Zakat Maal, Profesi, Emas, Fitrah, Fidyah, dan Pertanian.
- **Yasin & Tahlil**: Teks lengkap untuk amaliyah rutin.
- **Kitab Maulid**: Al-Barzanji, Ad-Diba'i, Simthud Durar, dan lainnya.
- **Ziarah Wali**: Daftar makam wali/ulama dengan integrasi Google Maps dan doa ziarah.
- **Wirid & Hizib**: Kumpulan Hizib Nahdliyah, Ratib Al-Haddad, dll.

### 📅 Kalender & Konten
- **Kalender Hijriah & Jawa**: Konversi tanggal Masehi ke Hijriah lengkap dengan Pasaran Jawa.
- **Konten Edukasi**: Artikel Islami, NUpedia (Ensiklopedia Tokoh), Khutbah Jumat, dan Video Kajian.
- **Kalam Quotes**: Galeri kutipan mutiara dari para ulama dan hadits.

### ⚙️ Personalisasi
- **Mode Gelap (Dark Mode)**: Mendukung kenyamanan mata saat malam hari.
- **PWA Ready**: Dapat diinstal di layar beranda HP seperti aplikasi native.
- **Sinkronisasi**: Riwayat bacaan Al-Qur'an terakhir tersimpan secara otomatis.

## 🚀 Teknologi yang Digunakan

- **Frontend**: React 18, Vite, TypeScript
- **Styling**: Vanilla CSS (Premium Mobile-First Design)
- **Icons**: Lucide React
- **Routing**: React Router DOM
- **State Management**: React Context API
- **APIs**:
  - AlAdhan API (Jadwal Shalat)
  - Al-Quran Cloud API (Teks & Terjemah)
  - OpenStreetMap Nominatim (Geolokasi)

## 🛠️ Cara Menjalankan Project

1. **Clone/Buka Folder**:
   Buka terminal di direktori project.

2. **Install Dependensi**:
   ```bash
   npm install
   ```

3. **Jalankan Development Server**:
   ```bash
   npm run dev
   ```

4. **Akses Aplikasi**:
   Buka [http://localhost:5173](http://localhost:5173) di browser Anda.

## 📱 Mobile-First Design
Aplikasi ini dioptimalkan untuk tampilan smartphone (lebar 380px - 430px). Untuk pengalaman terbaik, gunakan **Responsive Mode** di browser Anda atau buka langsung melalui smartphone.
